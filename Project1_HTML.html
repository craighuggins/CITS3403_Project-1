History

HTML stands for 'Hypertext Markup Language’ and is the standard language for documents displayed in a web browser. In its early stages, the internet was designed to distribute information across a noncentralized network of computers in times of emergency. Thus, the original tools for finding and viewing information on the internet were unsophisticated. In the early 1990’s, the number of organizations and people utilizing the internet was growing at a rapid rate, hence the need for easier access to resources and an enhanced display of the information.

HTML was invented by Tim Berners-Lee, a contractor at CERN when he proposed an Internet-based hypertext system for researchers to use and share documents. Berners-Lee wrote the software in late 1990.


HTML Versions Release Date Developments
1.0 1993 The first official version didn't have many features, other than getting some simple text onto the web.
2.0 1995 Introduced the general structure of an overarching <HTML> element containing the two main components <HEAD> and <BODY> (see the structure section for details), which was retained in all future versions. This version added key features such as structural elements for paragraphs, headings, ordered and unordered lists, as well as support for forms that allowed users to input and submit data.
3.2 1997 Supported new features such as text-flow around images, image maps, and tables. Greater control of layout was provided via the <DIV> element and class attribute, which allowed a page to be easily separated into different sections.
4.01 1999 Presentational features from HTML 3.2 were deprecated in favour of separate stylesheets, encouraging the separation of a documents structure from its presentation (see the CSS page for details).
5.0 2014 HTML5 added several new elements for explicit representation of structure in documents: <section>, <main>, <article>, <aside>, <header>, and <footer>.
The formerly used Adobe plug-in for video was dropped in favour of new <audio> and <video> elements, which removed the need for format-specific plug-ins, providing consistent player functions independent of browser platform.
Many new API's were made available for creating interactive applications.

All HTML Pages have the same basic structure, as can be seen in the example to the right. The structure is defined by a series of elements which tell the browser how to display the content. Elements are represented by tags, each of which has an opening and corresponding closing tag. Every HTML document has a root HTML node <html>, and two child nodes <head> and <body>.

Inside the head section, metadata is defined for the HTML document. This data is not displayed by the browser, but defines important information for the page to run. For example, a page title is added using the <title> tag. To see the title of this page, look at the current tab in your browser. Style sheets and Scripts are added here too, either internally or linked externally.

The <body> section of the web-page contains all content that is visible to the user of the website. This content is delivered via HTML elements:

Paragraphs

Used to display blocks of text on a page. For example, all of this text is considered a paragraph.

Headers

Headers are used to give a document structure. Users will skim read pages via its headings. The words Paragraphs and Headers are headers.

Tables

Tables are used to display sets of data on a page. They consist of Headers (<th>), Rows (<tr>), and Data cells (<td>). An example can be found in the history section of this page.

Lists

A list is a collection of items that may have no specific order, (<ul>), or a numbered sequence (<ol>). An example of an ordered list can be found in the Table of Contents on each page.

Div

The div tag is used to divide a document into sections. It is useful for laying out a webpage, and as a container to group html elements to style or perform specific tasks with via Javascript.

Images

Austria

Images are defined by the <img> tag, and require two attributes: src and alt. 'src' provides the source location for an image, and 'alt' provides text to be displayed in case the image is missing or cannot be loaded.
The image to the right was attached to its containing div element with the code: <img src="images/austria-mountains.jpeg" alt="Austrian Town Scenery">

Links

Links are fundamental to HTML as it provides navigation between pages. It is defined by the <a> tag, which has an 'href' attribute providing the destination.
For example, the following link will take you to the source website of the image shown on the right: Mountains and Cities of Austria

Links are not necessarily text-based. HTML elements can also be used as clickable link objects. For example, click on the image to go straight to the original-size source picture.
